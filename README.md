# FocusWrite

A focused writing environment built with React and Vite.

## Setup

```bash
npm install
```

## Scripts

- **`npm run dev`** – Start dev server
- **`npm run build`** – Production build
- **`npm run preview`** – Preview production build
- **`npm run lint`** – Run ESLint

## Stack

- React 19
- Vite 6
- React Router DOM
- Firebase
- Nanoid
